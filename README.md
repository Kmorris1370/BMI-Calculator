# __BMI-Calculator__
A calculator that accepts height and weight input then calculates BMI. Created using Java, Netbeans and Jframe. <br>

## __General Introduction__ 
This project is a simple BMI calculator with an interface. <br>

## __Features__ 
- Add height in feet/inches and weight in pounds to calculate BMI. <br>

## __Technologies__ 
Project created with:
- Java version: 24
- Netbeans IDE 25
- Java Swing <br>

## __How to Run__ 
To run this project, execute the BMICalculator.java file on any IDE with Java installed.<br>
